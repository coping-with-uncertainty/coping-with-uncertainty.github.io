# Data for Chapter 8: Simulating Random Variables and Simulation-based Inference

This page lists the datasets used in the R scripts for this chapter.

## Datasets

### Hartz4_2020.RDS and Hartz4_2022.RDS
- **Description**: Datasets containing social security (Hartz IV) statistics for the years 2020 and 2022.
- **Files**: [[Download 2020]](../data/Hartz4_2020.RDS) \| [[Download 2022]](../data/Hartz4_2022.RDS)
- **Usage**: Used in `permutation.r` for permutation testing.

## Scripts in this Chapter
- **approximate-bayes.R**: Illustrates Approximate Bayesian Computation.
- **permutation.r**: Performs permutation tests using Hartz IV data.
- **boot-classification.R**: Demonstrates bootstrap classification.
