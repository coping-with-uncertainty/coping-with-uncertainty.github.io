# Data for Chapter 4: Maximum Likelihood Estimation

This chapter primarily uses synthetic data generated directly within the R scripts. No external data files are required.

## Scripts in this Chapter
- **likelihood.R**: Demonstrates likelihood functions.
- **likelihood-principle.R**: Illustrates the likelihood principle.
- **poisson-regression.R**: Fits Poisson regression models on simulated sales data.
