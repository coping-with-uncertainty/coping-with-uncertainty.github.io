# Data for Chapter 3: Learning and Estimating

This chapter primarily uses synthetic data generated directly within the R scripts. No external data files are required.

## Scripts in this Chapter
- **credibility-intervall.R**: Calculates credibility intervals using simulated data.
- **konfidenzintervall.R**: Calculates confidence intervals using simulated data.
- **ks-test.R**: Performs Kolmogorov-Smirnov tests.
- **mendel.R**: Analyzes Mendel's laws with categorical data.
