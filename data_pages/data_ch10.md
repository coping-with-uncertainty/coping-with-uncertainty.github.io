# Data for Chapter 10: Inference in Extreme Data

This page lists the datasets used in the R scripts for this chapter.

## Datasets

### temp.dat
- **Description**: Historical temperature records used for extreme value analysis.
- **File**: [[Download]](../data/temp.dat)
- **Usage**: Used in `extreme-value.R` for GEV modeling.

## Scripts in this Chapter
- **extreme-value.R**: Analyzes extreme temperatures using Generalized Extreme Value (GEV) distributions.
- **censreg.R**: Performs censored regression analysis.
