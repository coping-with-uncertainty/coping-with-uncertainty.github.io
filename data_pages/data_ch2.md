# Data for Chapter 2: Uncertainty

This chapter primarily uses synthetic data generated directly within the R scripts. No external data files are required.

## Scripts in this Chapter
- **randomgenerator.R**: Generates random numbers for analysis.
- **kullback-leibler.R**: Computes KL divergence using simulated distributions.
- **p-value.R**: Performs p-value simulations.
- **simul-unknown-input.R**: Simulates unknown input scenarios.
