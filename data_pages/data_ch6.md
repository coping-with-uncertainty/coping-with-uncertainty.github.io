# Data for Chapter 6: Regression

This page lists the datasets used in the R scripts for this chapter.

## Datasets

### students.txt
- **Description**: A dataset containing student information for regression analysis.
- **File**: [[Download]](../data/students.txt)
- **Usage**: Used in `regression.R` to fit linear models.

## Scripts in this Chapter
- **regression.R**: Fits linear regression models on student data.
- **logistic-regression.R**: Demonstrates logistic regression using synthetic data.
- **logit.R**: Illustrates logit models.
