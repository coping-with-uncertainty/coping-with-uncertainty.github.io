# Data for Chapter 9: Bayesian Inference

This chapter primarily uses synthetic data generated directly within the R scripts. No external data files are required.

## Scripts in this Chapter
- **bayes-beta.R**: Illustrates Beta distributions in Bayesian inference.
- **Bayes-small-n.R**: Discusses Bayesian methods for small samples.
- **Gibbs-Normal.R**: Implements a Gibbs sampler for normal distributions.
- **zero-inflation.R**: Fits zero-inflated Poisson models.
